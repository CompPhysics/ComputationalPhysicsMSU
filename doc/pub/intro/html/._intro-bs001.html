Overview of first week

  • Tuesday (most likely):
    1. Presentation of the course, aims and content
    2. Introduction to C++ programming and numerical precision. Exercises for first week.
    3. Numerical differentiation and loss of numerical precision (chapter 3 lecture notes)
  • Computer lab: Thursday (most likely)
    1. The first two weeks we focus on simple programming tasks, start to look at project 1 and to set up Github and Qt Creator as one possible IDE. This week we discuss how to set up git and obtain a github account.
